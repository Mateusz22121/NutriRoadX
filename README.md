# Healthy Travel Food

Healthy Travel Food is an offline-first Android application designed to help users make healthier food choices while traveling, commuting, or working away from home.

The application is intended as a supportive tool that provides nutritional information and helps users make more informed decisions. It does **not** replace medical advice, a doctor, or a registered dietitian, and it does not prescribe or enforce any specific diet.

## Features

* Works completely offline
* Offline product database
* Barcode scanning
* Product search
* Nutritional information
* Fast and easy-to-use interface
* No user accounts
* No personal data collection
* No internet connection required

## Privacy

Your privacy is a priority.

The application:

* does not require user registration,
* does not collect personal information,
* does not track user activity,
* does not send any data to external servers.

Everything works locally on your device.

## Installation

The application is distributed as an APK file.

1. Download the latest APK release from this repository.
2. Transfer it to your Android device if necessary.
3. Enable installation from unknown sources (if required by your Android version).
4. Install the APK.
5. Launch the application.

## Future Plans

The application is designed with an architecture that allows future online features while keeping the core experience offline.

Possible future additions include:

* application updates via the internet,
* downloadable product database updates.

These features are intended solely to improve the application. User accounts, cloud synchronization, analytics, and personal data collection are **not** planned.

## Disclaimer

Healthy Travel Food is an informational tool intended to support healthier eating habits. It does not provide medical advice, diagnose medical conditions, or replace consultation with qualified healthcare professionals. Users should always rely on professional medical guidance when making health-related decisions.
